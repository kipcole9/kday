## Introduction

`kday` provides functions to return the date of the first, last or nth day of the week on, nearest, before or after a given date.

## Kday Installation

Add `kday` to your `deps` in `mix.exs`.

```elixir
def deps do
  [
    {:kday, "~> 1.0"}
    ...
  ]
end
```
